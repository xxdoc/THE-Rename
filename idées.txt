Support for FLAC, MpegPlus and vqf files.
allmusic.com support: search and export album information from the www.allmusic.com server to tags. 
Freedb support: get tag information for full MP3 albums in seconds using the CDDB1 Internet server
M3u playlists: create m3u playlists using the unique Quick Save Playlist feature.
